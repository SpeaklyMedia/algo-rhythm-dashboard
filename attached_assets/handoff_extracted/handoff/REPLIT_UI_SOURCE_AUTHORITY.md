# Replit UI Source Authority

## Authority Order
Use these local sources as layered authority. Do not treat them as equal.

1. `strategy_dashboard_replit` current app and `STRATEGY_DASHBOARD_INDEX.json`
   - Role: runtime truth, page contract, data wiring, required/optional file behavior
2. ThetaFrame
   - Role: lane organization, hero-summary-first page structure, semantic status system
3. Modal-Nodal / chat.mrksylvstr.com
   - Role: dark premium shell, low-glare surface language, pill-navigation and signal-accent posture
4. Yuki
   - Role: roadmap and selectable-function module layout, dashboard route framing
5. Flipper local scaffold
   - Role: fail-closed tone, local-first operational language, blunt control posture

## What Each Source Owns
### ThetaFrame
- `Plus Jakarta Sans` body-family direction
- calm hierarchical lane layout
- hero-summary-first page rhythm
- semantic status colors: ready / warn / danger / info

### Modal-Nodal
- dark neutral field
- cyan-primary signal accent
- restrained warm amber secondary emphasis
- sticky topbar / pill navigation behavior
- premium but product-focused shell tone

### Yuki
- modular route/page model
- roadmap-like grouping of functions and surfaces
- selectable-card treatment for grouped decision areas

### Flipper local scaffold
- local-only phrasing
- explicit failure states
- simple, trustworthy operational affordances

## Prohibited Misreads
- Do not use Flipper as the visual brand system.
- Do not copy Yuki’s anime-neon pink/purple palette literally.
- Do not make Modal-Nodal’s marketing shell more important than the review data.
- Do not flatten ThetaFrame’s hierarchy into a generic table-only dashboard.
